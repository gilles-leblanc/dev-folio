RandomNameGenerator :: Ruby application

Description

A Ruby program that can parse plain text files to generate probability tables and then generate random names using these tables.

Here are some sample names generated by the program when using input data taken from Greek mythology:
Ados, Heria, Lestia, Caemis, Catosesos, Hersa, Amede, Meliasa, Kraera, Celespa, Anyssa, Demine, Agra, Uros, Dikia, Arcos, Hemesiste.

Information


Tech: Ruby, RSpec
Algo: Markov chain
